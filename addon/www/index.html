README
======

Script:
string stdout;string stderr;system.Exec("/usr/local/addons/soundtouch/state.tcl IP COMMAND ARGUMENT",&stderr,&stdout);

Commands:
  • touch
  • volume

Arguments for the touch command:
  • PLAY
  • PAUSE
  • STOP
  • PREV_TRACK
  • NEXT_TRACK
  • THUMBS_UP
  • THUMBS_DOWN
  • POWER
  • MUTE
  • VOLUME_UP
  • VOLUME_DOWN
  • PRESET_1
  • PRESET_2
  • PRESET_3
  • PRESET_4
  • PRESET_5
  • PRESET_6
  • AUX_INPUT
  • SHUFFLE_OFF
  • SHUFFLE_ON
  • REPEAT_OFF
  • REPEAT_ONE
  • REPEAT_ALL
  • PLAY_PAUSE

Argument for the volume command is the new volume as number. if there is no argument the current volume will be fetched from the device.

CCU Variables:
BOSE_POWER_IP - Logic Variable - System is on or off
BOSE_VOLUME_IP - Number - Current Volume
BOSE_STATE_IP - String - Current Device State
